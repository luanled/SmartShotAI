# SmartShot: AI Camera Positioning & Framing

## Overview
SmartShot is an AI-powered mobile photography guidance system that helps users capture aesthetically pleasing photos in real-time. The system combines **Image Aesthetic Assessment (IAA)** with **Deep Reinforcement Learning (DRL)** to provide intelligent, actionable camera positioning guidance before a photo is taken.

The mobile app provides a live camera interface with real-time visual overlays (arrows, text) that guide users toward optimal compositions. The system learns from user interactions through a human-in-the-loop feedback mechanism, enabling personalized guidance that adapts to individual preferences over time.
